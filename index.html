BearMaps

I independently developed a browser-based map application of the East Bay that supports zooming and routing using A* algorithm, based on the Open Street Maps project and Apache Maven system.

Scheme Interpreter

This project is an Interpreter written in Python for the entire language of Scheme, a dialect of Lisp. The Interpreter parses the source code and performs its behavior directly; translates source code into an intermediate representation and immediately executes this and explicitly execute stored code made by the interpreter system. I additionally utilized tail recursion to increase storage efficiency, and macros to allow the language itself to be extended by the user.

Gravity Physics Simulator
