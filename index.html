Microsoft Hackathon Project: "Captionary" Computer Vision for Accessibility

Developer

When somebody with a visual impairment searches on Bing (or Google), they may not be able to tell what each image is. To solve this problem, we created an image analyzer feature for Bing Images that reads out loud an accurate description of what is in each image displayed.

For example, a photo might be described as “Barack Obama and Michelle Obama standing on a stage,” or “a black dog in a field of purple flowers.” This is compatible with both image upload and retrieving a link by hovering over an image. We improved upon the Azure Cognitive Services Computer Vision API, and utilized NLP to improve accurace.
